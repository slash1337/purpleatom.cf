Commands

There are all commands that Purple Atom has and can be used by its users.

  • MODERATION COMMANDS

    10
  • k.ban [mention] ─ Bans the mentioned user with a specific reason
  • k.kick [mention] ─ Kicks the mentioned user with a specific reason
  • k.mute [mention] [time in minutes] ─ Mutes the mentioned user
  • k.unmute [mention] ─ Unmutes the mentioned user
  • k.purge [amount of messages] ─ Purges an amount of messages (100 max.)
  • k.prune [amount of messages] ─ Prunes an amount of messages (100 max.)
  • k.ads [on/off] ─ Ads protection (no links on chat, you can post links only if you are an admin)
  • k.softban [mention] ─ Bans and unbans a user
    • See more...
  • k.lockdown [time in minutes] ─ Blocks texting in that channel so that only administrators can text. (BETA)
  • k.leaveg ─ Leaves the server (GUILD OWNER ONLY)

  • MUSIC COMMANDS

    9
  • k.play [link/name] ─ Plays a video(autojoin, no summon command needed)
  • k.clearqueue ─ Clears the per-server queue
  • k.leave ─ Tells the bot to leave the server
  • k.skip ─ Skips a song
  • k.pause ─ Pauses the playback
  • k.resume ─ Resumes the playback
  • k.loop ─ Loops the current played song
  • k.search [name] ─ Searches for a song on youtube
    • See more...
  • k.np ─ Shows what's playing in that moment

  • INFO COMMANDS

    17
  • k.serverinfo ─ Shows server's informations
  • k.userinfo [mention] ─ Shows user's informations
  • k.userperms [mention] ─ Shows user's guild permissions
  • k.id [mention] ─ Shows user's id
  • k.urban [your word here] ─ Urban Dictionary Search
  • k.avatar [mention] ─ Shows mentioned user's avatar and avatar id
  • k.ping ─ Shows my and API's ping
  • k.user ─ Shows my informations (no credit card numbers)

  • k.version ─ My current version

  • k.swstats [username] ─ Shows Hypixel SkyWars stats
  • k.mcserver [ip or dns] ─ Shows details about a minecraft server
  • k.check [ip or host] ─ Checks if the provided ip or host is online
  • k.hostinfo [ip or host] ─ Shows details about the provided ip or host
  • k.uptime ─ Shows my uptime
